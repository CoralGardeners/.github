# Coral Gardeners

**Revolutionizing ocean conservation and creating a global movement to save the reef.**

[Website](https://www.coralgardeners.org) · [CG Labs](https://cglabs.coralgardeners.org)

---

**CG Labs** is the technology arm of Coral Gardeners, a nonprofit organization working to restore and protect coral reefs. Coral Gardeners combines field-based restoration with data, monitoring technology, and community engagement. Our team is based in Moorea, French Polynesia, with restoration sites and partnerships all over the world.

This GitHub organization hosts the software behind our reef monitoring platform, restoration data pipeline, and member community tools.
